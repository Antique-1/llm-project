## 프로젝트 내용


### 1. 프로젝트 주제
* 인터넷 강의 음성 내용을 AI를 이용해 자동으로 요약하고, 그 요약문을 기반으로 문제를 자동 생성하는 시스템을 구현하는 것을 목표로 한다
* 기존 모델에서 부족한 부분을 탐색하고 추가적인 기능을 넣어 차별점을 주어 모델을 작성

  
### 2. 프로젝트 개발 배경 및 목표
* 온라인 강의 플랫폼의 확산으로 인해 요즘 온라인 강의를 통해 공부를 하는 사람들이 많아졌습니다. 이러한 강의를 듣는 많은 사람들의 학습 효율 상승을 위해
  AI를 이용한 요약과 문제 생성을 통해 내용을 복습하고 확인하면 좋겠다고 생각해 해당 프로젝트를 진행하게 되었습니다


  
### 3. 해당 프로젝트 관련 문헌
* [Du et al. (2017) — Learning to Ask: Neural Question Generation for Reading Comprehension](https://aclanthology.org/P17-1123/?utm_source=chatgpt.com)
  * 이 연구는 자연어 문장에서 자동으로 질문을 생성하는 신경망 기반 모델을 처음으로 제안. Sequence-to-Sequence 구조(Encoder–Decoder)를 이용하여 입력 문장을 이해하고, 그로부터 적절한 질문을 생성하도록 학습하였음.
    이 논문은 이후 T5, BART, GPT 계열 모델이 교육용 문제 자동생성에 활용되는 데 기반이 되었음 
* [Gonzalez et al. (2023) — Automatically Generated Summaries of Video Lectures: May Enhance Students’ Learning Experience](https://learninganalytics.upenn.edu/ryanbaker/ACL_2023.pdf?utm_source=chatgpt.com)
  * 이 연구는 자연어처리 기반 요약 알고리즘을 사용해 강의 텍스트를 압축하고, 학생들이 핵심 내용을 빠르게 복습할 수 있도록 했다. 실험 결과, 요약된 강의 내용을 제공받은 학생 그룹이 더 높은 학습 만족도와 이해도를 보였음
    -> 해당 문헌 내용을 근거로 AI를 이용한 강의 내용 요약이 학습 효율 향상에 도움이 된다는 것을 알 수 있었음
* [Multiple‑Choice Question Generation Using Large Language Models](https://arxiv.org/abs/2506.04851?utm_source=chatgpt.com)
  * 다양한 LLM(Llama 2, Mistral, GPT-3.5 등)을 비교하여 객관식 문제를 생성하여 비교 결과를 평가한 논문. 해당 모델 평가에서는 GPT가 현재 기준에서 가장 나은 성과를 보여주고 있음 -> 해당 내용을 근거로서 AI 생성 모델로 GPT를 채택하게 되었음

## 모델 개발 및 구현 과정

* 인터넷 강의 스크립트 또는 텍스트 자료로부터 자동으로 요약 문장을 도출하고, 이를 기반으로 객관식 문제(MCQ)를 생성하며, 각 문제에 대한 해설까지 자동으로 산출하는 AI 기반 교육 보조 시스템을 개발하는 것을 목표로 합니다
* 사전 학습된 OpenAI GPT 모델을 그대로 활용하는 제로샷(Zero-shot) 방식으로 문제 생성 기능을 수행하며, 별도의 추가 학습은 사용하지 않을 에정 -> 각 기능별 파인튜닝 하기위해 필요한 데이터셋이나 학습시간이 너무 많이 필요하고, gpt모델 자체 성능이 좋아 파인튜닝 없이 사용해도 성능이 충분하다고 생각하였음

### 1. 개발 범위

* 텍스트 입력(파일 또는 직접 입력)
* 텍스트 전처리 및 문맥 구조 정리
* GPT 모델 기반 요약 생성
* GPT 모델 기반 객관식 문제 생성(4지선다)
* 난이도 조절(초급/중급/고급) 기능
* 정답 및 오답 해설 자동 생성
* Streamlit 기반 웹 인터페이스로 결과 제공

---

### 2. 사용 모델 및 데이터

* 사용 모델
  * gpt-4o (openai)

* 데이터셋
  * 사용자 입력 데이터(강의 스크립트, 노트, 텍스트 파일 등)
  * 고정된 사전 학습 데이터셋을 사용하지 않으며, 입력 자료에 따라 결과가 동적으로 변함
  * 제로샷 기반이므로 별도의 학습용 데이터셋 구축이 필요하지 않음
